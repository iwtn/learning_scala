## 面白かったところ

- 関数がすべて Function0 〜 Function22 までのトレイトの無名サブクラスのインスタンス

```scala
val add = new Function2[Int, Int, Int]{
  def apply(x: Int, y: Int): Int = x + y
}

add: (Int, Int) => Int = <function2>
```
